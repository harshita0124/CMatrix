# CMatrix

The project aims to build a website for the matrix operations.

Following matrix operations will be done on the matrices:
- Addition (2 matrices)
- Subtraction (2 matrices)
- Multiplication (2 matrices)
- Inverse (1 matrices)
- Adjoint (1 matrices)
- Transpose (1 matrices)
- Determinant (1 matrices)
- Trace (1 matrices)
